After downloading, create a new folder within your resources directory and place the 'style.css, script.js, index.html, fxmanifest.lua, and client.lua' files inside it. Create a subfolder named 'media' for your assets; rename the background image to 'bg.jpg' and the music files to 'music1.mp3' and 'music2.mp3'. To start the script, type ensure [folder_name] or start [folder_name] in the live console or via the F8 menu,If it doesn't work, you can disable 'qb-loadingscreen'.
